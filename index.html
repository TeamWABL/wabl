Wabl

Wheeled Autonomous Balancing Limb - Inverted Pendulum Controller

View project on GitHub

This is an undergraduate engineering project at Taylor University.

Problem Statement

The goal of this project is to design and build a wheeled inverted pendulum, balancing about a single axis and limited to one-dimensional translational movement. Such a pendulum ought to stabilize in both angle and position within a reasonable time frame as well as minimize power consumption.

Proposal

This project will involve the use of an LQR control system to drive an electric motor in order to stabilize an intrinsically unstable system. The desired states (pendulum angle, position, and rotational as well as translational velocities) will be acquired and desired inputs computed in discrete time. By making use of state feedback and modern control theory, the full system will be implemented digitally.
